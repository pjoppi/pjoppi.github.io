# Privacy Policy

**Last updated:** March 26, 2026

This Privacy Policy explains how **Biome Checker** (the "Bot") collects, uses, and handles data. The Bot is developed by **hxz0**.

## Information We Collect

The Bot is very simple and collects only the minimum data needed to function:

- Your Discord User ID and username (when you run the command)
- The biome verification command you send

No message content is scanned automatically. The Bot only reacts when you actively use the verification command.

## How We Use the Information

- We use the command input to communicate with **your personal RDP instance** and perform the biome verification.
- The Bot does **not** store message content or command history long-term.
- No data is used for advertising, analytics, or shared with third parties.

## Data Storage and Retention

We do **not** store any personal data or command logs persistently.  
The only data kept is basic configuration required for the Bot to operate. All verification requests are processed in real-time and then discarded.

## Sharing of Data

Data is only sent between:
- Discord (to receive your command)
- My own RDP instance (to verify the biome)

No data is forwarded to any external services or other users.

## Your Rights

You can remove the Bot from your server or stop using the command at any time.  
If you want any stored data deleted (though almost none exists), contact us.

## Contact Us

Questions or data deletion requests:  
Join the support server: https://discord.gg/solsniper

We may update this policy. Continued use of the Bot means you accept any updates.
