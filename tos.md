# Privacy Policy

**Last updated:** March 26, 2026

This Privacy Policy explains how **macromeow** (the "Bot") collects, uses, and handles data. The Bot is developed by **hxz0**.

## Information We Collect and Process

To provide its features (leaking webhook scanner, message forwarding, and macro channel creation), the Bot processes the following data:

- Discord User IDs, usernames, and avatars of message authors
- Discord Server ID and server name
- Channel IDs and channel names (the Bot scans all channels it has access to)
- **Message content**, including text, embeds, attachments, and links
- Configuration data (e.g., source/destination channel mappings, macro channel settings)

## How We Use the Information

We use this data **only** to operate the Bot:

- Scan channels in real-time for leaking webhooks
- Forward messages from selected channels to other channels **within the same Discord server**
- Create and manage macro channels as configured by server administrators
- Maintain the Bot's functionality

We do **not** use message content for any other purpose such as training AI, advertising, or selling data.

## Sharing of Data

Message content is forwarded **only inside the same Discord server** where the Bot is added.  
We do **not** send your data to any third parties, external servers, or sell/share it in any way.

## Your Rights

- Server administrators can remove the Bot at any time to stop all scanning and forwarding.
- You can request access to or deletion of any stored data (such as configuration or logs) by contacting us.
- We respect applicable privacy laws (including GDPR where relevant).

We may update this Privacy Policy occasionally. Continued use of the Bot after changes constitutes acceptance of the updated policy.
